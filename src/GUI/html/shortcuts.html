Shortcuts

This section lists various shortcut commands you can type or press with the keyboard when constructing various proofs.

General

Command Explanation Keyboard shortcuts
Add row Add an empty row beneath a current row. Enter
Delete row Delete a current row. Note, the formula field in the proof panel needs to be empty to remove a row. If it is not empty, then pressing backspace until the formula is gone will delete the row. backspace
Add Assumption box Opens an assumption box at a specific row when inputting the shortcut in the rule field. as , TODO
Add row outside of assumption box When inside an assumption box, pressing enter will add a row in that current box. This command is used to jump outside of the box and add a row outside of it. Alt+Enter
Add Premise Adds a Premise in the rule field. TODO, pr
